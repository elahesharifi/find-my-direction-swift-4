ایده کلی
---
شما با استفاده از این برنامه می توانید پس از وارد کردن مبدا و مقصد دلخواه، نزدیکترین مسیر با استفاده از مترو و یا اتوبوس را مشاهده کنید. همچنین زمان تخمینی و خطوط مسیر نیز بر روی نقشه مشخص میشوند.

---

</li>
<li>

</li>
</ol>
